# 每日工作与学习进度

这个文档用于持续记录两条主线：

1. 学习进度
2. 项目进度

使用原则：

1. 每天只追加一条记录，不改历史结论
2. 每条记录尽量控制在 5 到 10 分钟内能写完
3. 先写结果，再写问题，最后写明天计划
4. 学习内容必须尽量映射到项目模块，避免只学不做
